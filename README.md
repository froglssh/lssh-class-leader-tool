# 班級幹部小老師選取獎勵程式

這是一個給導師使用的單頁網頁工具，可匯入學生名單、設定班級幹部與小老師、進行指派與匯出。

## 使用方式

開啟 GitHub Pages 網址即可使用。資料主要儲存在瀏覽器本機，不會自動上傳學生名單。

- 線上主程式：https://froglssh.github.io/lssh-class-leader-tool/
