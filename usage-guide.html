班級幹部小老師選取獎勵程式(codex優化版) 使用說明

臺北市麗山高中林獻升老師製作。本文協助導師快速完成學生名單匯入、幹部與小老師職務設定、手動或隨機指派，以及結果匯出。

開啟線上版程式 查看 GitHub 專案

一、程式用途

本程式用於導師安排班級幹部與各科小老師。老師可以匯入學生名單，使用預設職務清單，或自行新增職務，再透過手動選擇或隨機分配完成安排，最後將結果下載為 Excel 或 PDF。

線上版網址：https://froglssh.github.io/lssh-class-leader-tool/

二、開始使用

  1. 開啟線上版程式，或在電腦中開啟本機 HTML 檔。
  2. 進入「學生名單」分頁，先建立學生資料。
  3. 進入「任務清單」分頁，建立幹部與小老師職務。
  4. 進入「分配與檢視」分頁，進行手動指定或隨機分配。
  5. 進入「匯出 / 匯入」分頁，下載 Excel、PDF 或 JSON 備份。

建議每次正式分配前，先確認學生名單與職務清單是否正確。若使用共用電腦，完成後可按「重設全部」清除本機資料。

三、匯入學生名單

方法 1：Excel 匯入

  1. 在「學生名單」分頁中，點選「Excel 匯入」區塊的檔案選擇按鈕。
  2. 選擇學生名單 Excel 檔，支援 .xlsx、.xls、.csv。
  3. 按「讀取班級清單」。如果檔案內有多個班級，程式會產生班級下拉選單。
  4. 在「選擇班級」中挑選要匯入的班級。
  5. 確認是否勾選「匯入時先清空目前學生名單」。一般情況建議保持勾選。
  6. 按「匯入所選班級」。學生名單會出現在下方表格。

方法 2：手動新增或批次貼上

如果沒有 Excel，也可以直接新增單筆學生，或在批次新增欄位中貼上名單。批次新增每行一位，可使用 座號,姓名 格式，例如：

1,王小明
2,李小華
3,陳大文

四、建立幹部與小老師職務

進入「任務清單」分頁後，可以切換「幹部任務」或「小老師任務」。

生成常用幹部 使用表單預設的幹部名單，例如班長、副班長、風紀股長、衛生股長、保健股長等。
生成常用小老師 使用表單預設的小老師名單，例如國文、英文、數學、物理、化學、生物、歷史、公民、藝能小老師等。
自訂職務 可在單筆新增或批次新增欄位中輸入自訂職務，例如「設備股長」「午餐股長」等。

若發現職務多打或不需要，可在任務表格右側按「刪除」。

五、分配學生

手動指定

  1. 進入「分配與檢視」分頁。
  2. 選擇分配類型：「幹部」或「小老師」。
  3. 在每個職務右側的下拉選單中選擇學生。
  4. 如需取消某一職務，按該列的「清除」。

隨機分配

  1. 確認學生名單與職務清單都已建立。
  2. 選擇分配類型。
  3. 若不希望同一位學生在同一類型中重複擔任多個職務，請保持「同一類型避免重複指派同一人」勾選。
  4. 按「隨機分配未指派」。程式只會補上尚未指派的職務。

若只想處理某一個職務，可使用每列右側新增的「隨機」按鈕，直接為該列從目前可用學生中隨機指定一位。已指派的職務也可以再次按下「隨機」重新分配；若勾選避免重複，程式會自動排除同類型中已被其他職務使用的學生。

六、儲存、匯出與備份

功能 用途 建議使用情境
儲存到本機 將目前資料存在瀏覽器本機儲存空間。 同一台電腦稍後繼續編輯。
下載 Excel 輸出 .xlsx，包含分配結果與學生名單。 交給行政、整理紀錄、後續修改。
下載 PDF 開啟列印視窗，可另存為 PDF。 列印、傳閱、存成正式紀錄。
下載 JSON 完整保存程式資料，可再匯入程式。 備份未完成的工作，或跨裝置繼續編輯。

七、常見問題

Excel 名單匯入後看不到班級怎麼辦？

請確認 Excel 內有清楚的「班級、座號、姓名」欄位，或每個班級區塊附近有班級名稱。如果檔案格式太特殊，可先另存為 .xlsx 或整理成「班級、座號、姓名」三欄後再匯入。

學生名單匯入後有舊資料混在一起怎麼辦？

匯入前請勾選「匯入時先清空目前學生名單」。若已混在一起，可按「清空名單」後重新匯入。

下載 CSV 會亂碼嗎？

新版已移除 CSV 匯出，改用 Excel 與 PDF，避免中文在不同軟體中出現亂碼。

學生名單會上傳到網路嗎？

不會。程式主要在瀏覽器中執行，學生名單與分配資料不會自動送到伺服器。若按下載，檔案會儲存在使用者自己的電腦。

八、建議工作流程

  1. 先匯入該班學生名單。
  2. 在任務清單中生成常用幹部與小老師。
  3. 刪除不需要的職務，或新增班上特有職務。
  4. 先手動指定已確定人選，再用隨機分配補齊剩下職務。
  5. 檢查分配結果，下載 Excel 與 PDF 保存。
  6. 若尚未完成，另外下載 JSON 作為下次繼續編輯的備份。
